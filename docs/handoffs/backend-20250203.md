# Backend handoff — 2025-02-03

## Scope

Address audit gaps so the app can extend beyond homelab use. Backend changes only.

## Decisions

1. **DB connection:** Add `@contextmanager def db_connection()` in vault_db; use in main.py via `with db_connection() as conn:` so every route uses it. No FastAPI dependency (keeps sync handlers simple).
2. **Setup/signup:** Add `_key_for_user(conn, username, password) -> tuple[int, bytes]` helper in main.py; use in post_setup and post_signup.
3. **Session persistence:** In session.py `_persist_load`, catch `(sqlite3.Error, json.JSONDecodeError, ValueError, KeyError)` specifically; log and re-raise others.
4. **Recovery key:** Add `vault_db.generate_and_store_recovery(conn, user_id, key) -> str` so main.py stays thin.
5. **Pydantic max_length:** username 64, title 256, notes 4096, url 2048, recovery q/a 256 each.
6. **post_vault_reset:** Wrap os.remove in try/except OSError; log and raise HTTPException 500 on failure.
7. **GET /recovery/questions:** Add rate limit check (reuse _check_auth_rate_limit; needs Request param).
8. **Lock audit:** Retrieve session before delete to get user_id; pass to audit.log_event("lock", user_id=user_id).
9. **Config docstring:** Fix to say "Validates config at startup (e.g. in lifespan)".
